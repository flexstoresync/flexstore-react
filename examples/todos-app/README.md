# Todos app example

Reference layout for a Vite + React + TypeScript app using `@flexstore/react`.

Copy this structure into your project, or browse the files in
[`packages/react/examples/todos-app`](../../packages/react/examples/todos-app) on GitHub.

## Run (monorepo)

```bash
cd developer/tests/mytodo
yarn
yarn dev
```

Set `.env` from `.env.example` and register your origin in the FlexStore dashboard
(or use [self-hosted FlexStore](https://github.com/flexstoresync/flexstore-self-host)).

**Related:** [flexstore-core](https://github.com/flexstoresync/flexstore-core) · [flexstore](https://github.com/flexstoresync/flexstore) monorepo
